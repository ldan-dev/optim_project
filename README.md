# Concepts
### Virtual environment
- python base:
```bash
py -3.13 -m venv venv
# venv\Scripts\activate   # En Windows
source venv/bin/activate  # En Linux/macOS
pip install -r requirements.txt
```

- conda/ miniconda:
```bash
conda create -n optim_project python=3.13 -y
conda activate optim_project
python -m pip install -r requirements.txt
```

**Note**: Python version: **3.13.11**
